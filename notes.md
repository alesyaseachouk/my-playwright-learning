 File/Folder | What it does |
|---|---|
| `package.json` | Lists dependencies and npm scripts |
| `package-lock.json` | Locks exact versions of dependencies |
| `node_modules/` | Stores all installed npm packages |
| `playwright.config.ts` | Configures Playwright settings like browsers and timeouts |
| `tests/` | Folder where all test files live |
| `.gitignore` | Tells Git which files and folders to ignore |
